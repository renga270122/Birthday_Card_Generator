## 3-Step Setup

### Step 1: Install Dependencies
```bash
cd thirupugazh-app
npm install
```

Wait for packages to install (takes ~1-2 minutes)

### Step 2: Start the App
```bash
npx expo start
```

You'll see a QR code in your terminal!

### Step 3: Open on Your Phone
- Open **Expo Go** app on your phone
- **iOS**: Point your Camera at the QR code
- **Android**: Scan the QR code with Expo Go app
- The app will load on your device!

## Common Issues?

**Can't see QR code?**
- Try running: `npx expo start --tunnel`

**Phone and computer on different WiFi?**
- Make sure both devices are on the same network
- Or use tunnel mode: `npx expo start --tunnel`

**Dependencies installation failed?**
- Delete `node_modules` folder
- Run `npm install` again
