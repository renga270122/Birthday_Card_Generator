# Audio Files for Birthday Card Generator

## Required Audio Files

This directory should contain royalty-free Happy Birthday instrumental music files in multiple versions.

### Files to Add:

1. **happy-birthday-classic.mp3** - Classic instrumental version (30 seconds)
2. **happy-birthday-jazz.mp3** - Jazz instrumental version (30 seconds)  
3. **happy-birthday-piano.mp3** - Piano instrumental version (30 seconds)

### Sources for Royalty-Free Music:

- **YouTube Audio Library** (https://studio.youtube.com/channel/UC/music)
- **Free Music Archive** (https://freemusicarchive.org/)
- **Pixabay Music** (https://pixabay.com/music/)
- **Incompetech** (https://incompetech.com/)
- **Bensound** (https://www.bensound.com/)

### Format Requirements:

- Format: MP3 (for broad compatibility)
- Bitrate: 128-192 kbps (balance quality/file size)
- Duration: 25-35 seconds
- File size: < 1MB each (for fast loading)

### Copyright Note:

All audio files must be:
- Public domain, OR
- Creative Commons licensed (with attribution), OR
- Royalty-free with commercial use allowed

**Important**: The "Happy Birthday to You" song is in the public domain as of 2016 in the United States. However, specific recordings may still be copyrighted. Use instrumental versions or create your own recordings.

### Temporary Alternative:

For testing purposes, the application will work without audio files and show a message that music files need to be added.
